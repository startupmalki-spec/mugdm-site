# ZATCA Placeholder Corpus

This directory holds short, publicly-summarised references to ZATCA
(Zakat, Tax and Customs Authority) rules used by Mugdm's RAG pipeline.

**Important:** These files intentionally contain only brief public summaries
with prominent citation links back to the original source. They are NOT
substitutes for the official regulation text. When authoritative answers are
needed, follow the "Source:" links below to zatca.gov.sa.

Add new `.md` files here and re-run `scripts/ingest-rag-corpus.ts`.

Source portal: https://zatca.gov.sa/en/Pages/default.aspx
